Minisampler

What is this web?

Minisampler is a web tool intended to allow musicians to create basic rythms and tunes using samples.

how do I use it?

This tool is very simple. First select the samples you want as instruments, you can do this by selecting the sample name in the dropdown list of the matching instrument, or click the instrument button in the middle right block and the click the sample button in the samples tab list at the bottom. Both methods will link a sample to a track. Next, select which instrument should play and at what time in the pattern matrix in the middle left block. Play with the speed and track length and that's it.
